Our Model


We made a model of a Hydropower plant! this uses DC motors to generate electricity. basically, if you put a current through the motor, then it'll spin. But if you spin the motor, then it'll create it's own current.


At the time of writing this website we don't know if we're going to use batteries as an example on this model or if we're going to put water down a pipe.
